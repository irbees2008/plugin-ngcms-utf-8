# NG Captcha - Универсальный плагин защиты от ботов

Универсальный плагин для Next Generation CMS, объединяющий три популярных провайдера капчи в одном решении.

## Поддерживаемые провайдеры

- **Google reCAPTCHA v3** - невидимая капча с оценкой действий пользователя
- **Cloudflare Turnstile** - современная альтернатива reCAPTCHA без раздражающих проверок
- **Яндекс SmartCaptcha** - российский сервис с поддержкой GDPR

## Возможности

- Единый интерфейс для всех провайдеров
- Переключение между провайдерами в настройках
- Защита регистрации, комментариев и обратной связи
- Гибкие настройки для каждого провайд ера
- Проверка только для гостей (опционально)

## Требования

- NGCMS Next Generation
- Активированный плагин `ng-helpers`
- Ключи от выбранного провайдера капчи

## Установка

1. Скопировать папку `ng-captcha` в `engine/plugins/`
2. Активировать плагин в панели администратора
3. Выбрать провайдера и указать ключи в настройках

## Получение ключей

### Google reCAPTCHA v3
https://www.google.com/recaptcha/admin/create

### Cloudflare Turnstile
https://dash.cloudflare.com/

### Яндекс SmartCaptcha
https://cloud.yandex.ru/services/smartcaptcha
